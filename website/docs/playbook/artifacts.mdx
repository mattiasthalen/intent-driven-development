# Artifacts

Instead of Jira boards and burndown charts, IDD teams maintain these living documents:

**Intent Queue** — A prioritized list of intent documents. Not a backlog — it's curated, not dumped into. Items that haven't been touched in 30 days get archived automatically.

**Decision Log** — A chronological record of every ACCEPT / REDIRECT / RETHINK / PARK decision with rationale. Searchable. This IS the project history.

**Context Files (CLAUDE.md)** — The project's institutional memory, optimized for AI consumption. Conventions, patterns, anti-patterns, taste decisions.

**Design Principles** — Accumulated taste decisions. "We prefer composability over convenience." "Error messages should suggest fixes." "No magic — everything is explicit."

**Convergence Dashboard** — A simple tracker showing how many loops each intent took to reach ACCEPT. Trends here reveal process problems.
