# The Core Loop

IDD replaces sprints with a continuous loop that runs at whatever cadence the problem demands — sometimes minutes, sometimes days.

| Phase | Action | Key Question |
|---|---|---|
| **1. INTENT** | Define what should exist and why | Is the desired outcome clear? |
| **2. GENERATE** | AI produces code, designs, options | Does the AI have sufficient context? |
| **3. EVALUATE** | Human reviews against intent + taste | Does this meet our acceptance criteria and feel right? |
| **4. DECIDE** | Accept, reject, or redirect | What's the right next step? |
| **5. LEARN** | Update context and constraints | What did we learn that improves the next loop? |
